# My Site

Site pessoal para centralizar meus contatos e destacar meus principais projetos. Com objetivo de servir como cartão de visita virtual, seja para processos de seleção ou para qualquer pessoa que queira me conhecer melhor.

Acesse em [www.r4f4siqueira.com](https://www.r4f4siqueira.com)

## Tecnologias utilizadas

- HTML
- CSS
- Git
- GitHub Actions
- Firebase Hosting

## Boas práticas implementadas

- Código organizado e comentado
- Controle de versão com _Git_
- Deploy automático com _GitHub Actions_
- Controle de features através de `branchs` e `Pull Requests`
- Preview de deploy em cada `PR` mergeada para `main`
